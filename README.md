# LevelForge 🚀

Aplicación web para gestionar metas y tareas con una interfaz tipo feed inspirada en redes sociales.

## 🧠 Concepto

LevelForge convierte tus metas en un sistema visual tipo timeline, donde cada tarea es como un “post” que puedes seguir, completar y enriquecer con recursos.

## ✨ Features

* 📌 Sistema de metas (boards)
* 🧾 Tareas tipo feed (estilo Twitter)
* ⏳ Contador regresivo por tarea
* ✅ Marcar tareas como completadas
* 📂 Recursos adicionales por tarea
* 👤 Avatar global del usuario
* 🎨 Personalización con imágenes

## 🛠️ Tecnologías

* HTML
* CSS
* JavaScript (Vanilla)
